# Capítulo 2 — Fundamentação Teórica

## 1. Revisão Bibliográfica
Apresenta os principais conceitos e trabalhos relacionados ao tema: técnicas de SRS (Spaced Repetition Systems), abordagens de processamento de linguagem natural (NLP) para geração de exemplos/frases, síntese de fala (TTS) e o uso de modelos de linguagem (LLMs) para diálogo.

## 2. Técnicas de Memorização e SRS
- Princípios cognitivos por trás do SRS (curva do esquecimento, espaçamento ideal).
- Algoritmos comuns (Anki, SM-2, variantes adaptativas).

## 3. Processamento de Linguagem Natural e Geração de Conteúdo
- Extração de definições e exemplos a partir de APIs/Corpora.
- Técnicas para evitar geração de frases incorretas ou ofensivas.

## 4. Modelos de Linguagem para Diálogo
- Arquitetura dos LLMs modernos, fine-tuning vs prompting.
- Estratégias para controlar vocabulário ativo do modelo (prompt engineering, context windows, retrieval).

## 5. Síntese de Voz e Acessibilidade
- APIs e técnicas de TTS, formatos de áudio, latências aceitáveis.
- Considerações de acessibilidade e internacionalização.

## 6. Engenharia de Software para Aplicativos Móveis
- Padrões para arquitetura móvel (clean architecture, MVVM/MVC), teste e CI/CD.

## 7. Trabalhos Relacionados
- Resumo de projetos/estudos que integraram SRS + diálogo automático.

Cada seção pode conter referências bibliográficas e links para artigos, RFCs e documentação de APIs utilizadas.
